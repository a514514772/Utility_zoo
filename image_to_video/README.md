# Image-to-Video

## Quick Start
```
python image_to_video.py trans_list.txt
```
## Usage 
```
usage: image_to_video.py [-h] [--output_path OUTPUT_PATH] [--width WIDTH]
                         [--height HEIGHT]
                         img_list

Convert a series of images to a single video.

positional arguments:
  img_list              List describing the path to images

optional arguments:
  -h, --help            show this help message and exit
  --output_path OUTPUT_PATH
                        Path to the output file
  --width WIDTH         width of images
  --height HEIGHT       height of images
```
